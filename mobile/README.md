# 📱 Hay Mobile App

## Setup Instructions

### 1. Install Dependencies
```bash
cd mobile
npm install
```

### 2. Run on Simulator/Device

#### iOS
```bash
npm run ios
```

#### Android
```bash
npm run android
```

#### Web
```bash
npm run web
```

### 3. Build for Production

#### Android
```bash
npm run build-android
```

#### iOS
```bash
npm run build-ios
```

## Features

✅ Real-time Chat
